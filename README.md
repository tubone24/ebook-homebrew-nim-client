> Make pdf file which you use e-books by take in some image files such as jpeg, png and gif.

# Setup

Use Nim and Nimble, so you should install nim (>=0.20.2)

```
$ nimble build -d:ssl
```

# Run

for LINUX
```
$ ./bin/ebook_homebrew_nim_client
```
